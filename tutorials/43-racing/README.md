# Racing — Tutorial

Build a complete **Racing** game from scratch using TypeScript and HTML5 Canvas.

**Difficulty:** Advanced
**Episodes:** 7
**Time:** ~1h 45min total
**Prerequisites:** [Top-Down Shooter](../42-topdown-shooter/README.md)

## What You'll Build

A top-down racing game with acceleration, braking, and drifting around a track. Race against AI opponents, complete laps, and post the fastest time.

## Concepts You'll Learn

- Vehicle physics (acceleration, braking, steering, friction)
- Track rendering from waypoints or tile maps
- Lap detection using checkpoint triggers
- AI racers following the track path
- Mini-map display of the full track

## Episodes

| # | Title | Time | What You'll Build |
|---|-------|------|-------------------|
| 1 | [Project Setup & Track Rendering](./step-1.md) | ~15min | Draw a race track from a defined path |
| 2 | [Car Rendering & Controls](./step-2.md) | ~15min | Draw a car, steer with keys, accelerate and brake |
| 3 | [Vehicle Physics](./step-3.md) | ~15min | Friction, drift, off-track slowdown |
| 4 | [Track Boundaries & Collision](./step-4.md) | ~15min | Keep the car on the track, wall collisions |
| 5 | [Checkpoints & Lap Counting](./step-5.md) | ~15min | Checkpoint gates, lap counter, lap timer |
| 6 | [AI Opponents](./step-6.md) | ~15min | Computer cars that follow the racing line |
| 7 | [Mini-Map & Polish](./step-7.md) | ~15min | Mini-map overlay, position display, finish screen |
