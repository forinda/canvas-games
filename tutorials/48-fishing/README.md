# Fishing — Tutorial

Build a complete **Fishing** game from scratch using TypeScript and HTML5 Canvas.

**Difficulty:** Intermediate
**Episodes:** 5
**Time:** ~1h 15min total
**Prerequisites:** [Tower Defense](../47-tower-defense/README.md)

## What You'll Build

A relaxing fishing game where you cast your line, wait for a bite, then reel in the catch using a tension-based minigame. Collect different fish species and fill your tackle box.

## Concepts You'll Learn

- Cast-and-reel mechanics with power gauge
- Tension/line-strength minigame (keep in the green zone)
- Fish AI with randomized bite timing and fight behavior
- Underwater scene rendering with parallax layers
- Collection/catalog system for caught fish
