# Basketball — Tutorial

Build a complete **Basketball** shooting game from scratch using TypeScript and HTML5 Canvas.

**Difficulty:** Intermediate
**Episodes:** 5
**Time:** ~1h 15min total
**Prerequisites:** [Doodle Jump](../15-doodle-jump/README.md)

## What You'll Build

A basketball shooting game where you aim and launch a ball toward a hoop. Drag to set angle and power, then release to shoot. Score points for baskets, with bonus points for swishes.

## Concepts You'll Learn

- Projectile motion with arc trajectories (parabolic physics)
- Drag-to-aim input with angle and power visualization
- Rim and backboard collision detection
- Ball-through-net detection (scoring zone)
- Trail rendering for ball path
