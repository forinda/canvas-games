# Balloon Pop — Tutorial

Build a complete **Balloon Pop** game from scratch using TypeScript and HTML5 Canvas.

**Difficulty:** Beginner
**Episodes:** 4
**Time:** ~1h total
**Prerequisites:** [Whack-a-Mole](../08-whack-a-mole/README.md)

## What You'll Build

Colorful balloons float upward from the bottom of the screen. Click to pop them before they escape off the top. Different colors are worth different points, and special balloons grant bonuses.

## Concepts You'll Learn

- Spawning and moving objects across the screen
- Object pooling for performance
- Click-to-destroy mechanics with hit testing
- Particle burst effects on pop
- Varying object properties (speed, size, color, score)

## Episodes

| # | Title | Time | What You'll Build |
