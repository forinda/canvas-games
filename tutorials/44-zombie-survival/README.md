# Zombie Survival — Tutorial

Build a complete **Zombie Survival** game from scratch using TypeScript and HTML5 Canvas.

**Difficulty:** Advanced
**Episodes:** 7
**Time:** ~1h 45min total
**Prerequisites:** [Racing](../43-racing/README.md)

## What You'll Build

A top-down survival game where zombies swarm from all directions. Shoot them, build barricades, and survive as many nights as possible. Scavenge resources between waves.

## Concepts You'll Learn

- Horde AI with flocking/swarming behavior
- Barricade placement and destruction mechanics
- Resource management (ammo, health, materials)
- Day/night cycle affecting gameplay
- Spatial partitioning for many-entity collision
