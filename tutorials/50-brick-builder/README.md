# Brick Builder — Tutorial

Build a complete **Brick Builder** construction toy from scratch using TypeScript and HTML5 Canvas.

**Difficulty:** Intermediate
**Episodes:** 4
**Time:** ~1h total
**Prerequisites:** [Idle Clicker](../49-idle-clicker/README.md)

## What You'll Build

A creative building game where you place colored bricks on an isometric grid to construct whatever you imagine. Snap bricks together, change colors, and rotate pieces to build structures.

## Concepts You'll Learn

- Isometric grid rendering (2D to isometric projection)
- Snap-to-grid brick placement
- Layer/height management for stacking bricks
- Color palette and brick type selection
- Save and load constructions with JSON serialization
