# Doodle Jump — Tutorial

Build a complete **Doodle Jump** platformer from scratch using TypeScript and HTML5 Canvas.

**Difficulty:** Intermediate
**Episodes:** 5
**Time:** ~1h 15min total
**Prerequisites:** [Helicopter](../14-helicopter/README.md)

## What You'll Build

An endless vertical-scrolling platformer. A character auto-bounces off platforms while you steer left and right. Reach higher and higher while avoiding gaps and hazards.

## Concepts You'll Learn

- Vertical auto-scrolling camera that follows the player
- Platform generation with random spacing
- Auto-bounce mechanics on landing
- Screen wrapping (exit left, appear right)
- Platform types (normal, breakable, moving)

## Episodes

| # | Title | Time | What You'll Build |

| 4 | [Platform Types & Hazards](./step-4.md) | ~15min | Add breakable, moving, and spring platforms |
