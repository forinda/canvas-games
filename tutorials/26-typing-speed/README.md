# Typing Speed — Tutorial

Build a complete **Typing Speed** test game from scratch using TypeScript and HTML5 Canvas.

**Difficulty:** Intermediate
**Episodes:** 5
**Time:** ~1h 15min total
**Prerequisites:** [Fruit Ninja](../25-fruit-ninja/README.md)

## What You'll Build

A typing speed test that displays text on screen and measures how fast and accurately you type it. See your words-per-minute (WPM), accuracy percentage, and track improvement over time.

## Concepts You'll Learn

- Real-time keyboard input matching against displayed text
- Words-per-minute (WPM) and accuracy calculation
- Character-by-character highlighting (correct, incorrect, pending)
- Canvas text measurement and word wrapping
- Progress bar and live statistics display

## Episodes

| # | Title | Time | What You'll Build |
|---|-------|------|-------------------|
| 1 | [Project Setup & Text Display](./step-1.md) | ~15min | Render a paragraph of text with word wrapping on canvas |
| 2 | [Typing Input & Character Matching](./step-2.md) | ~15min | Capture keystrokes, highlight correct/incorrect characters |
| 3 | [WPM & Accuracy Tracking](./step-3.md) | ~15min | Calculate live WPM and accuracy as the user types |
| 4 | [Timer & Results Screen](./step-4.md) | ~15min | Countdown timer, final results with stats breakdown |
| 5 | [Word Bank & History](./step-5.md) | ~15min | Multiple text passages, personal best tracking, difficulty modes |
