# Idle Clicker — Tutorial

Build a complete **Idle Clicker** game from scratch using TypeScript and HTML5 Canvas.

**Difficulty:** Intermediate
**Episodes:** 5
**Time:** ~1h 15min total
**Prerequisites:** [Fishing](../48-fishing/README.md)

## What You'll Build

An incremental idle game where you click to earn currency, buy upgrades that generate income automatically, and prestige to reset with multiplied earnings. Watch numbers grow exponentially.

## Concepts You'll Learn

- Exponential cost scaling and big-number formatting
- Idle/automatic income generation per tick
- Upgrade and multiplier systems
- Prestige/rebirth mechanic for long-term progression
- Offline progress calculation on return

## Episodes

| # | Title | Time | What You'll Build |

| 1 | [Project Setup & Click Counter](./step-1.md) | ~15min | Draw a clickable button and a currency counter |

| 4 | [Prestige System](./step-4.md) | ~15min | Reset progress for a permanent multiplier bonus |
| 5 | [Offline Progress & Polish](./step-5.md) | ~15min | Calculate earnings while away, save/load with localStorage |
