# Canvas Game Arcade — Tutorial Series

A step-by-step YouTube tutorial series building **50 canvas games** from scratch. Each game builds on concepts from the previous one, progressing from beginner to advanced.

## Series Overview

- **Total episodes:** ~270 steps across 50 games
- **Skill progression:** Absolute beginner to advanced game developer
- **Tech stack:** TypeScript, HTML5 Canvas, Vite (no frameworks)
- **Architecture:** SOLID principles introduced gradually

## How to Use This Series

1. **Follow in order** — each game builds on concepts from previous ones
2. **Code along** — each step has complete code you can type and run
3. **Experiment** — each step ends with suggested challenges
4. **Review** — each game README links all steps with summaries

## Step Format

Each `step-N.md` follows this structure:

```
## Step N: Title
### What you'll learn
### Prerequisites (from previous steps)
### The code
### How it works (explanation)
### Try it (run instructions)
### Challenge (extend it yourself)
```
