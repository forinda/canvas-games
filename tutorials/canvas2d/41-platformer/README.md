# Platformer — Tutorial

Build a complete **Platformer** game from scratch using TypeScript and HTML5 Canvas.

**Difficulty:** Advanced
**Episodes:** 7
**Time:** ~1h 45min total
**Prerequisites:** [Particle Sand](../40-particle-sand/README.md)

## What You'll Build

A side-scrolling platformer with running, jumping, enemies, and collectibles. Navigate through multiple levels, stomp enemies, and reach the flag at the end.

## Concepts You'll Learn

- Platformer physics (run, jump, wall slide, coyote time)
- Tile-based level loading from a map array
- Enemy AI with patrol and chase behaviors
- Side-scrolling camera following the player
- Collectibles, checkpoints, and level completion
