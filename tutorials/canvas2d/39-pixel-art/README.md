# Pixel Art — Tutorial

Build a complete **Pixel Art** editor from scratch using TypeScript and HTML5 Canvas.

**Difficulty:** Intermediate
**Episodes:** 4
**Time:** ~1h total
**Prerequisites:** [Card Battle](../38-card-battle/README.md)

## What You'll Build

A pixel art drawing tool with a zoomable grid canvas. Pick colors from a palette, paint pixels, use tools like fill and eraser, and export your creations as PNG images.

## Concepts You'll Learn

- Pixel-level Canvas manipulation with imageData
- Drawing tools (pencil, eraser, fill bucket)
- Color palette and color picker implementation
- Zoom and pan on a grid-based canvas
- Exporting canvas content as a downloadable PNG

## Episodes

| # | Title | Time | What You'll Build |
|---|-------|------|-------------------|
| 1 | [Project Setup & Pixel Grid](./step-1.md) | ~15min | Draw a zoomable pixel grid on the canvas |
| 2 | [Drawing & Erasing Tools](./step-2.md) | ~15min | Click and drag to paint pixels, eraser tool |
| 3 | [Color Palette & Fill Tool](./step-3.md) | ~15min | Color picker, palette display, flood-fill bucket tool |
| 4 | [Export & Polish](./step-4.md) | ~15min | Export as PNG, undo/redo, grid toggle, zoom controls |
