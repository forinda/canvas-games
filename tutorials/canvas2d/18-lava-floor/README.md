# Lava Floor — Tutorial

Build a complete **Lava Floor** platformer from scratch using TypeScript and HTML5 Canvas.

**Difficulty:** Intermediate
**Episodes:** 5
**Time:** ~1h 15min total
**Prerequisites:** [Golf](../17-golf/README.md)

## What You'll Build

A platformer where the floor is lava and it's rising! Jump between platforms to stay above the rising lava. Survive as long as possible while collecting gems for bonus points.

## Concepts You'll Learn

- Rising hazard zone (lava) creating urgency
- Platform jumping with variable jump height
- Scrolling camera that follows the rising threat
- Collectible items on platforms
- Particle effects for lava bubbles
