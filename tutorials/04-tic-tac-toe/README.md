# Tic-Tac-Toe — Tutorial

Build a complete **Tic-Tac-Toe** game from scratch using TypeScript and HTML5 Canvas.

**Difficulty:** Beginner
**Episodes:** 5
**Time:** ~1h 15min total
**Prerequisites:** [Memory Match](../03-memory-match/README.md)

## What You'll Build

A two-player Tic-Tac-Toe game with an optional simple AI opponent. Draw Xs and Os on a 3x3 grid and detect wins, losses, and draws.

## Concepts You'll Learn

- Turn-based game logic
- Drawing shapes (X and O) with Canvas paths
- Win-condition checking (rows, columns, diagonals)
- Simple AI opponent (random + basic strategy)
- Game-over detection and replay

## Episodes

| # | Title | Time | What You'll Build |
|---|-------|------|-------------------|
| 1 | [Project Setup & Board Rendering](./step-1.md) | ~15min | Draw the 3x3 grid lines on the canvas |
| 2 | [Placing Marks & Turns](./step-2.md) | ~15min | Click cells to place X or O, alternate turns |
| 3 | [Win & Draw Detection](./step-3.md) | ~15min | Check rows, columns, and diagonals for a winner |
| 4 | [AI Opponent](./step-4.md) | ~15min | Add a computer player with basic strategy |
| 5 | [Score Tracking & Polish](./step-5.md) | ~15min | Track wins/draws, add winning-line animation, restart |
