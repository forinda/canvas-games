# Golf — Tutorial

Build a complete **Golf** game from scratch using TypeScript and HTML5 Canvas.

**Difficulty:** Intermediate
**Episodes:** 6
**Time:** ~1h 30min total
**Prerequisites:** [Basketball](../16-basketball/README.md)

## What You'll Build

A 2D mini-golf game with multiple holes. Aim and shoot the ball across terrain with slopes, walls, and obstacles. Get the ball in the hole in as few strokes as possible.

## Concepts You'll Learn

- Ball rolling physics with friction and damping
- Surface collision and wall bouncing
- Aim-and-power stroke input
- Terrain and obstacle layout per hole
- Stroke counting and par scoring
